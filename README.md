# How to run tests
Step 1 - Install node on your computer.
Step 2 - Clone repo
Step 3 - Install project dependencies inside repo folder.
		npm  install 
Step 4 - Run Cypress with
		npx cypress open
Step 5 - Chose e2e
Step 6 - Open browser
Step 7 - Run desirable test (AT1, AT2, AT3)
